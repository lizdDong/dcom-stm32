# dcom-stm32

#### 介绍
STM32的外设和组件驱动集合

## 驱动包含有：
#### 1、片内外设驱动
	DIO - 支持输入滤波
	ADC - 采用DMA，支持双通道同步采样，支持卡尔曼滤波、部分中值滤波、部分最大值滤波等多种滤波形式
	DAC - 支持多种波形输出
	UART - 支持中断和DMA两种传输方式，采用fifo缓存机制
	CAN - 采用fifo缓存机制
	FLASH - 支持任意位置读写
	另外还有RTC、SPI、IWDG、PWM等等驱动模块
	
#### 2、组件
	FIFO - 支持读写超时机制，临界区保护机制
	LOG - 支持日志等级设置，支持输出文件名、行号、时间戳等信息，支持十六进制和二进制方式输出日志
	LED - 支持闪烁输出
	FSM - 有限状态机，支持入状态和出状态执行函数
	IAP - 在系统升级，可配合私有协议进行
